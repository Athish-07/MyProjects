## ⚙️ Getting Started

### Prerequisites
- Java 11 or higher
- Maven (for Spring Boot projects)
- IntelliJ IDEA or any Java IDE

### Running a Project

**For Spring Boot projects (Casino, Chat App v2.0):**
```bash
# Unzip the project
unzip <project>.zip && cd <project-folder>

# Run with Maven
./mvnw spring-boot:run
```

**For Console/Desktop projects (Hangman, Alarm Clock, User Registration):**
```bash
# Open in IntelliJ IDEA and run Main.java
# OR compile manually
javac src/Main.java
java -cp src Main
```

**For Airline Management System:**
```bash
# Deploy on Apache Tomcat server
# Import the project into Eclipse/IntelliJ as a Dynamic Web Project
```
